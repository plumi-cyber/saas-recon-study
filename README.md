# SaaS Landing Page Recon Study

A passive OSINT study of what public B2B SaaS companies expose in their landing pages — no logins, no probing, no testing of discovered endpoints.

Motivated by the @weezerOSINT ClickUp thread (April 2026), which demonstrated how much enterprise risk lives in plain JavaScript bundles.

**Status:** In progress. Setup phase.

## Scope
- 10 public B2B SaaS landing pages
- Observation only: page source, network tab, DevTools, public DNS, certificate transparency
- No interaction with any discovered endpoints, keys, or tokens
